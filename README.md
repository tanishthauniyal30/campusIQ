# CampusIQ

An iOS app built with SwiftUI that centralizes a student's academic life — CGPA, attendance, electives, and opportunities — into one place, powered by LLM integration for smart features.

## Features
- **Dashboard** — overview of academic status at a glance
- **Electives** — browse and manage elective choices
- **CGPA** — track and calculate CGPA
- **Attendance** — monitor attendance records
- **Opportunities** — discover relevant opportunities

## Tech Stack
- **Frontend:** Swift, SwiftUI, Xcode
- **AI/LLM Integration:** Groq API (Llama 3.3 70B) — migrated from Gemini API after hitting quota limits during development
- **Version Control:** Git/GitHub

## Status
Actively in development — login/sign-in and persistent data storage are in progress.

## Background
Built as the technical foundation for an application to Graphic Era University's Apple/Infosys-backed iOS Development program.
